# Changelog

## [0.0.15] - 2026-04-25 (unreleased)

### Added
- キーバインド設定 UI（設定画面でのキーマッピング一覧表示・編集）
- デフォルトキーバインドをパブリック関数として公開
- キーバインドのテキストキー定義を追加
- EXIF メタデータ表示のサポート
- ズームアクションの追加（新しいズーム操作）
- `PreviousImageAction` の実装
- リソーステキストのオーバーライド機能の改善（デフォルト設定ディレクトリを考慮したサジェストパスを追加）
- ヘルプドキュメントの更新（キーバインド一覧表示の改善）
- ファイルアクション機能の強化と UI 統合
- `next_policy_directory` が zip および listed 仮想ルートを直接返すように修正
- ビューワーリクエスト ID 割り当てとプリロード管理の実装（`ViewerApp`）
- ビューワー機能・リソース管理のユニットテスト追加
- ファイルシステム操作・ソート・UI インタラクションの包括的テスト追加
- ビューワーモジュールを `dialogs.rs` / `navigation.rs` / `workers.rs` に分割リファクタリング
- `filesystem/function.rs` として独立したファイルシステム関数モジュールを追加

### Changed
- wml2 ライブラリを 0.0.21 → 0.0.22 に更新
- キーソートロジックの強化、予約バインドの条件を簡略化
- マウスイベントバインドの整理
- 右クリックメニューの修正
- 設定メニューのイメージセクションテキストを明確化
- Windows リソースコンパイル時のアイコンファイル名を修正（`wml2viwer` → `wml2viewer`）
- コードの可読性・保守性向上のためのリファクタリング
- テストフォーマットおよび構造の改善
- `.env` を `.gitignore` に追加

---

## [0.0.14] - 2026-04-17

### Added
- `path_classification.rs` 新規追加（パス種別判定ロジックを独立モジュール化）
- `wml2_formats.rs` 新規追加（wml2 フォーマット定義）
- `FilerUserRequest` enum による Filer のリクエスト状態管理
- Filer から兄弟コンテナ・ランダムコンテナへのブラウジングサポート
- ランダムファイル選択機能の追加
- ビューワーのナビゲーションアクション強化（Home/End キーによるエッジナビゲーション）
- `resolve_end_path` 関数の追加（エッジナビゲーションの改善）
- ビューワーソート順の設定
- Zip ファイルのウォーク処理の最適化
- 拡張子の自動収集機能
- `--bench` オプションの追加
- `benchlog.rs` によるベンチマークログ機能の追加
- `DisplayedPageState` に関連フィールドを統合したコンパニオン状態管理のリファクタリング
- プリロードキャッシュ機能の追加
- ドキュメント整備（`docs/modules.md`, `docs/state.md`, `docs/nexttodo.md`, `docs/followup.md`）
- CI/CD ワークフロー設定（`.github/workflows/release.yml`）

### Fixed
- Zip ナビゲーションのバグ修正
- スキャンフォルダのバグ修正
- Filer がファイルを更新したときにビューワーが新しいファイルを反映しないバグを修正
- ビューワーと Filer の連携バグを修正
- Home/End キーのフォルダスキャンメッセージの問題を修正
- マンガモードの問題修正
- wml2 0.0.20 更新：インデックス付き PNG の修正

### Changed
- wml2 ライブラリを 0.0.19 → 0.0.20 に更新
- バックグラウンドファイルシステムリクエスト処理を削除（リファクタリング）
- ファイルシステムの再初期化処理の改善
- リクエスト陳腐化チェックの実装
- コードの可読性・一貫性向上のためのリファクタリング
