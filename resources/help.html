wml2viewer Help

Mouse / Context Menu

  • Right-click on the viewer to open the context menu.
  • Direct menu items include Image Information, Settings, and About.
  • File actions are available from the same menu: Move, Copy, Delete, Rename.
  • Save Image offers format selection and opens the save dialog.

Dialogs

  • Image information opens as an information dialog (title: Information).
  • System operation results from Settings open with title Settings.
  • About text opens with title About.
  • Generic errors open in an alert dialog (title: Alert).

File Action Destinations

  • Move/Copy dialogs support both manual input and folder browsing (Browse...).
  • Settings > Navigation provides destination presets for: Move Folder1/Folder2 and Copy Folder1/Folder2.
  • Use Use as Move / Use as Copy to switch active destination slots.
  • After successful Move/Copy, the destination is saved to the active slot and persisted to config.
